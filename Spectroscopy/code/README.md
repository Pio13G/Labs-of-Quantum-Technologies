## Scripts

This file contains a brief recap on the use of the files inside this folder

|   File        | Description |
|---------------|-------------|
| doppler1      | Analysis of the first data taken with the doppler effect. can be re-used modifying the intervals used in the various interpolations |
| doppler2      | Analysis of the second data taken with the doppler effect. can be re-used modifying the intervals used in the various interpolations |
| subdop0_low   | Analysis of the file S1 data taken using saturated spectroscopy. Note that this one has one of the best dips. |
| subdop1_low   | Analysis of the file S1_BIG data taken using saturated spectroscopy. Since the time scale is big, the dips are formed by only few points |
